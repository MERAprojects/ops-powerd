ops-powerd
==========

What is ops-powerd?
-------------------
ops-powerd is a daemon that detects and reports power supply status.

What is the structure of the repository?
----------------------------------------
* src/ contains the source files for ops-powerd
* include/ contains the header files for ops-powerd
